# Week 1: Foundation & Setup

## Project Overview
**Project Name:** Environment & Hello World  
**Duration:** Week 1  
**Objective:** Install industry-standard DevOps tools and build a basic proof of concept with containerization and cloud deployment.

## Learning Objectives
- Set up a professional DevOps development environment
- Understand containerization with Docker
- Deploy a simple application to AWS
- Learn Infrastructure as Code (IaC) basics with Terraform
- Implement CI/CD fundamentals with GitHub Actions

## Tech Stack
- **Containerization:** Docker
- **Cloud Platform:** AWS (EC2, ECR)
- **Infrastructure as Code:** Terraform
- **Version Control:** Git & GitHub
- **CI/CD:** GitHub Actions
- **Programming:** Python/Node.js (for Hello World app)
